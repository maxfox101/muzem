# АИС «Быть воином — жить вечно»

Веб-приложение для музея «Самбекские высоты» на React + Vite + TypeScript + TailwindCSS.

## 🚀 Быстрый старт

```bash
cd hero-memorial-app
npm install
npm run dev
```

Откройте http://localhost:5173

## 📁 Структура проекта

```
src/
├── components/
│   ├── FormHero/          # Форма заявки
│   ├── AdminPanel/        # Админ-панель (облачное хранилище)
│   ├── ModeratorDashboard/# Модерация заявок
│   └── Subscription/      # Подписка на акции
├── pages/                 # Страницы (будут добавлены)
├── hooks/                 # React хуки
├── services/              # API сервисы
└── utils/                 # Утилиты
```

## ✅ Реализовано

- ✅ Форма заявки с валидацией (WCAG 2.1 AA)
- ✅ Автокоррекция ФИО (первая буква БОЛЬШАЯ)
- ✅ Календарь для дат (не text input)
- ✅ Админ-панель с настройками облачного хранилища
- ✅ Модераторский дашборд (3 действия: принять/отклонить/уточнить)
- ✅ Подписка на акции с кодом подтверждения
- ✅ Мобильная версия (mobile-first, от 320px)
- ✅ Все рамки border-2, контрастность 4.5:1, фокусные состояния

## 🎨 Стандарты доступности

- Контрастность текста/фона: минимум 4.5:1 (AA)
- Все интерактивные элементы: минимум 44×44px
- Фокусные состояния: border-2px, ring-4px
- Клавиатурная навигация
- Адаптивная верстка: 320px, 480px, 768px, 1024px, 1280px

## 📱 Мобильная версия

Проект использует mobile-first подход. Все компоненты адаптированы для экранов от 320px.
